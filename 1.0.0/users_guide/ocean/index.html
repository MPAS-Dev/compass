Ocean core¶

The ocean core in compass contains test groups and test cases for the MPAS-Ocean) model. For more information on MPAS-Ocean, see the most recent user’s guide for version 6. Currently, there are 4 test groups—baroclinic_channel, global_ocean, ice_shelf_2d, and ziso, each with several Test Cases. Many more ocean test cases are available through Legacy COMPASS (see legacy ocean test cases for a few examples).

Some helpful external links:

  • MPAS website: https://mpas-dev.github.io

  • MPAS-Ocean User’s Guide includes a quick start guide and description of all flags and variables.

  • Test cases (not requiring compass) for latest MPAS-Ocean release:

    • https://mpas-dev.github.io/ocean/releases.html

    • https://mpas-dev.github.io/ocean/release_6.0/release_6.0.html
