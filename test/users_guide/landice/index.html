Landice core¶

The landice core in compass contains test cases for the MPAS-Albany Land Ice (MALI) model. For more information on MALI, see the most recent Hoffman et al. (2018). Currently, there are 2 configurations—dome and enthalpy_benchmark, each with several Test Cases. Many more landice test cases are still available through Legacy COMPASS.

Some helpful external links:

  • MPAS website: https://mpas-dev.github.io

  • MPAS-Albany Land Ice Model User’s Guide v6.0 includes a quick start guide and description of all flags and variables.

  • Test cases (not requiring compass) for latest MPAS-Ocean release:

    • https://mpas-dev.github.io/land_ice/download.html

    • MPAS-Albany Land Ice v6.0 Test Cases
