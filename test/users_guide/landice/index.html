Landice coreÂ¶

The landice core in compass contains test cases for the MPAS-Albany Land Ice (MALI) model. Currently, there are not yet any test cases in this core and landice test cases are only available through Legacy COMPASS.

We hope to add landice test cases to the compass package in the very near future.
