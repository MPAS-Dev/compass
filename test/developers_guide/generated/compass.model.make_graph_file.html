compass.model.make_graph_file¶

compass.model.make_graph_file(mesh_filename, graph_filename='graph.info', weight_field=None)[source]¶

Make a graph file from the MPAS mesh for use in the Metis graph partitioning software

Parameters

mesh_filename (str) – The name of the input MPAS mesh file

graph_filenamestr, optional

The name of the output graph file

weight_fieldstr

The name of a variable in the MPAS mesh file to use as a field of weights
