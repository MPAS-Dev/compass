compass.io.download¶

compass.io.download(file_name, url, config, dest_path=None, dest_option=None, exceptions=True)[source]¶

Download a file from a URL to the given path or path name

Parameters

  • file_name (str) – The relative path of the source file relative to url and the destination path relative to dest_path (or the associated config option)

  • url (str) – The URL where file_name can be found

  • config (configparser.ConfigParser) – Configuration options used to find custom paths if dest_path is a config option

  • dest_path (str, optional) – The output path; either dest_path or dest_option should be specified

  • dest_option (str, optional) – An option in the paths config section defining an output path; either dest_path or dest_option should be specified

  • exceptions (bool, optional) – Whether to raise exceptions when the download fails

Returns

out_file_name (str) – The resulting file name if the download was successful
