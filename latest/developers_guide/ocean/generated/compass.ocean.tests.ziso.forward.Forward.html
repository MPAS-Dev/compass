compass.ocean.tests.ziso.forward.Forward¶

class compass.ocean.tests.ziso.forward.Forward(test_case, resolution, name='forward', subdir=None, with_analysis=False, with_frazil=False, long=False, with_particles=False)[source]¶

A step for performing forward MPAS-Ocean runs as part of ZISO test cases.

Variables

  • resolution (str) – The resolution of the test case

  • with_analysis (bool, optional) – whether analysis members are enabled as part of the run

  • with_frazil (bool, optional) – whether the run includes frazil formation

__init__(test_case, resolution, name='forward', subdir=None, with_analysis=False, with_frazil=False, long=False, with_particles=False)[source]¶

Create a new test case

Parameters

  • test_case (compass.TestCase) – The test case this step belongs to

  • resolution (str) – The resolution of the test case

  • name (str) – the name of the test case

  • subdir (str, optional) – the subdirectory for the step. The default is name

  • with_analysis (bool, optional) – whether analysis members are enabled as part of the run

  • with_frazil (bool, optional) – whether the run includes frazil formation

  • long (bool, optional) – Whether to run a long (3-year) simulation to quasi-equilibrium

  • with_particles (bool, optional) – Whether particles are include in the simulation
