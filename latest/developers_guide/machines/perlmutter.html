Perlmutter

For most machine-specific details (including config options and how to enable hyperthreading), see the User’s Guide under Perlmutter.

pm-cpu, gnu

If you’ve set things up for this compiler, you should be able to source a load script similar to:

source load_dev_compass_1.2.0-alpha.2_pm-cpu_gnu_mpich.sh

Then, you can build the MPAS model with

make [DEBUG=true] gnu-cray
