Cori¶

cori-haswell, intel¶

This is the default compass architecture and compiler on Cori. To activate the compass environment, load modules, and set appropriate environment variables, run this in the compass repo root:

source ./load/load_compass_env.sh -m cori-haswell

You don’t have to supply -m cori-haswell but it will prevent you from getting a warning that you might want cori-knl.

To build the MPAS model with

make CORE=landice intel-nersc

or

make CORE=ocean intel-nersc

cori-haswell, gnu¶

To activate the compass environment, load modules, and set appropriate environment variables, run this in the compass repo root:

source ./load/load_compass_env.sh -m cori-haswell -c gnu

As above, you don’t have to supply -m cori-haswell but it will prevent you from getting a warning that you might want cori-knl.

To build the MPAS model with

make CORE=landice gnu-nersc

or

make CORE=ocean gnu-nersc
