Chrysalis¶

intel¶

This is the default compass compiler on Chrysalis. If the environment has been set up properly (see compass conda environment, compilers and system modules), you should be able to source:

source load_dev_compass_1.0.0_chrysalis_intel_impi.sh

Then, you can build the MPAS model with

make intel-mpi

gnu¶

If you’ve set things up for this compiler, you should be able to:

source load_dev_compass_1.0.0_chrysalis_gnu_openmpi.sh

Then, you can build the MPAS model with

make gfortran
