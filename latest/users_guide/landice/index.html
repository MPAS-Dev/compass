Landice core¶

The landice core in compass contains test groups and test cases for the MPAS-Albany Land Ice (MALI) model. For more information on MALI, see Hoffman et al. (2018). Currently, there are 5 test groups—dome, enthalpy_benchmark, eismint2, greenland, and hydro_radial—each with several Test Cases. Many more landice test cases are available through Legacy COMPASS.

Some helpful external links:

  • MPAS website: https://mpas-dev.github.io

  • MPAS-Albany Land Ice Model User’s Guide v6.0 includes a quick start guide and description of all flags and variables.

  • Test cases (not requiring compass) for latest MALI release:

    • https://mpas-dev.github.io/land_ice/download.html

    • MPAS-Albany Land Ice v6.0 Test Cases
